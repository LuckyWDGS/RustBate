# Niagara VFX Artist Skill 使用指南

## 安装位置
✓ `C:\Users\QY\.claude\skills\niagara-vfx.md`

## 如何使用

### 1. 启动 Skill
在 Claude Code 中输入：
```
/niagara
```

这会启动一个新的对话会话，Claude 将扮演专业的 Niagara 特效艺术家角色。

### 2. 使用场景

#### 场景 A: 提供图片参考
```
/niagara

我想做一个类似这样的特效 [上传图片]
```

#### 场景 B: 文字描述
```
/niagara

我想做一个火焰爆炸特效，颜色是蓝色的，有强烈的冲击感
```

#### 场景 C: 分析现有特效
```
/niagara

这是我做的特效 [上传截图/视频]，帮我分析一下有什么可以改进的
```

#### 场景 D: 参数调整
```
/niagara

我的特效太亮了，而且粒子消失得太快，怎么调整？
```

#### 场景 E: 风格建议
```
/niagara

我想做一个魔法护盾特效，但不知道用什么颜色和形状
```

## Skill 提供的内容

### 1. 视觉分析 🎨
- 艺术风格识别
- 视觉基调分析
- 美学建议

### 2. 色彩方案 🌈
- 主色调、强调色、过渡色
- HSV/RGB/Hex 色值
- Color Curve 具体参数

### 3. 运动设计 ⚡
- 蓄力、爆发、消散三阶段设计
- 时间节点和参数值
- 非线性曲线配置

### 4. Niagara 配置 🔧
- Emitter Settings
- Particle Update Modules
- Renderer 选择
- Material Parameters
- 具体数值和范围

### 5. 实现建议 💡
- 分步实现指南
- 注意事项
- 性能优化建议

## 技巧

### 1. 提供清晰的描述
越详细越好：
- ✓ "蓝紫色的魔法护盾，有能量波纹，受击时产生涟漪"
- ✗ "做个护盾"

### 2. 提供参考图片
一图胜千言，直接上传参考图片效果最好

### 3. 说明使用场景
- "这是角色技能特效"
- "这是环境装饰特效"
- "这是 UI 特效"

### 4. 说明性能要求
- "移动端游戏，需要优化"
- "PC 端，可以复杂一些"

### 5. 迭代优化
可以多次对话调整：
```
/niagara
[第一次设计]

"颜色太暗了，能调亮一点吗？"
[调整后的参数]

"很好！但是运动太快了"
[再次调整]
```

## 注意事项

1. **新会话**: 每次使用 `/niagara` 都会开启新的对话会话
2. **图片支持**: 可以直接上传图片让 AI 分析
3. **参数范围**: AI 会提供参数范围，方便你微调
4. **物理合理**: 特效会符合基本物理规律（除非你特别说明）
5. **性能考虑**: AI 会在建议中考虑性能影响
